# Python Project: Video Game Sales Analysis Project

## Description
This project analyzes video game sales data to explore revenue trends across different regions and worldwide markets. The analysis focuses on understanding how game platforms, genres, publishers, and release years impact sales performance.

---

## Project Objectives
- Analyze regional sales performance:
  - North America (NA)
  - Europe (EU)
  - Japan (JP)
  - Global market
- Identify top-selling games and platforms
- Compare genre popularity across regions
- Explore publisher performance
- Visualize sales trends over time

---

## Dataset Information

| Column Name | Type | Description |
|------------|------|-------------|
| `rank` | int | Sales ranking |
| `name` | str | Game title |
| `platform` | str | Gaming platform (e.g., `Wii`, `NES`, `PS4`) |
| `year` | int | Year of release |
| `genre` | str | Game genre (e.g., `Sports`, `Platform`, `Racing`) |
| `publisher` | str | Publishing company |
| `na_sales` | float | North America sales in millions of USD |
| `eu_sales` | float | Europe sales in millions of USD |
| `jp_sales` | float | Japan sales in millions of USD |
| `global_sales` | float | Worldwide sales in millions of USD |
